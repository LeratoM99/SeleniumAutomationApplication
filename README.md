Description:
A professional Selenium automation framework built using Java, Maven, TestNG, Page Object Model (POM), Data-Driven Testing (Excel), ExtentReports, screenshot capture on failure, and CI/CD integration using GitHub Actions.

Key Features:

UI automation using Selenium WebDriver

TestNG framework with annotations and listeners

Page Object Model design pattern

Data-driven testing using Excel (Apache POI)

Automatic screenshots on test failure

Professional HTML test reports (ExtentReports)

Continuous Integration using GitHub Actions

Technologies Used:

Java

Selenium WebDriver

TestNG

Maven

Apache POI

ExtentReports

GitHub Actions

How to Run:

Clone the repo

Run: mvn clean test

View report: target/ExtentReport.html
